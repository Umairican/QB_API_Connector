Connect QuickBooks (One-time OAuth Consent)

A finance admin authorizes our QuickBooks app once. We request the read-only scope: com.intuit.quickbooks.accounting.

What happens

  1. Admin signs in to QuickBooks and approves access for Ashling Partners.
  2. QuickBooks returns a short-lived access token and a long-lived refresh token.
  3. We store only the refresh token securely and use it server-side to request temporary access tokens when running scheduled report pulls.

Data pulled

  • General Ledger
  • Accounts Receivable Aging
  • Accounts Payable Aging
  • Trial Balance
